Today we'll be looking at repetition. Perl provides a for loop for solving counting problems, along with a while and until loop. The until loop works like a while loop, except that it runs until the condition is true (syntactical sugar really). Let's look at a BNF for these three loop types:

LoopStatement ::= LoopClause StatementList
LoopClause    ::= for (Expression;Expression;Expression) { statementList }
| foreach identifier (Array) { statementList }
| while (Expression ) { statementList }
| until (Expression) { statementList }

Our particular problem is an example of a counting problem, in which we want to count from start to stop by intervals of increment, displaying the value of count and its logarithm. For counting problems, the Perl for loop provides the most convenient notation, but you'll practice that in this week's project so we'll use a while loop today.

In order to actually compute the logarithms, we'll use Perl's log() function. However, Perl's log() function computes the natural logarithm of the expression. To compute the log base 10 of a number we have to use basic algebra. To do this divide the natural log of the number by the natural log of the desired base.

We can use Perl's say statement to print a row of our table. But say will only display a single string. We could write out each piece of the row with separate say (or print) statments, but Perl allows you to insert variable names directly within strings. In addition to the direct variable insertion Perl allows concatenations of expressions (such as an algebraic expression that calculates a logarithm) to the end of a string simply by seperating the string and the expression by a comma (Perl also has a way to preform string interpolation by enclosing the function in curly braces and dereferencing the method like so ${\ log($number)}, but this issomething of a hack with the string parser and is not recognized as an offical method). Write a single statement to print out a row.

Then, all that's left to do is to increment our counting variable by the increment variable the user provided.
